Death Stranding

Watch Trailer

Death Stranding is a 2019 action game developed by Kojima Productions. It is the first game from director Hideo Kojima and Kojima Productions after their split from Konami in 2015.

The game is set in the United States following a cataclysmic event which caused destructive creatures to begin roaming the Earth.

Players control Sam Porter Bridges (Norman Reedus), a courier tasked with delivering supplies to isolated colonies and reconnecting them via a wireless communications network
